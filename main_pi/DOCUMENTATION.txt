это документация к главной raspberry pi. которая управляет отображением здоровья города, время которое осталось у участников


FILE hp.txt - это общее количество hp у всего города. путем выолнения скриптов на других малинах (air, chp, lcd) 
будет отниматся определённое количество hp и выводится на экран для удобного отображения в виде линии. нахождение файла - /home/main/hp.txt


FILE ssh_change_num - это файл который должен находится на каждой малине (air, chp, lcd) он создан для изменения хп на главной малине


FILE apsys.service находтся в /etc/systemd/system/apsys.service и преднозначен для актвации wifi раздачи на raspberry pi.
на main раздается wifi с ssid "main" и паролем "20182007".
к main подключаются все оставльные малины.


FILES ssh_change_num.sh и start_or_stop.py отвечают за включение, выключение светодиода(.py). А так же запускает bash скрипт,
который отнимает значение у hp.txt.
